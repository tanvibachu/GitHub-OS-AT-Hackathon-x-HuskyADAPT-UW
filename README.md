# Q Charms Accessories Documentation

## Overview
This file will describe the two sub-projects (bracelet extender, lanyard) within the Q Charms accessories project. Each section below will briefly describe the background for each, and then describe the hackathon design/prototype, referencing any relevant files.

These projects are designed to be used with Q charms, which are 3D printable charms that have raised Augmentative and Alternative Communication (AAC) icons on them. Q charms were developed by Woof3D and the Washington Assitive Technology Act Program (WATAP) at the University of Washington, so that a standard silicon bracelet can be used as a communication aid by sliding charms on. They were created based on Boardmaker charms. You can access different printable [Q charm designs on Thingiverse](https://www.thingiverse.com/thing:7155293/files)

Or, you can create your own Q charms designs using our Customized Q Charms project in this same repo!

### Bracelet Extender

#### Background
Q Charms are designed to be worn on a standard silicon bracelet. However, these bracelets are not very adjustable for different wrist sizes. Silicon bracelets that can be snap-adjusted already exist, but even these can be limited in size range - especially when being used with Q charms, which can significantly add to the circumference needed for a comfortable bracelet. An easy-to-use bracelet extender was requested to use with the Q charms snap bracelets. An example of these snap bracelets are these [snap bracelets on Amazon](https://www.amazon.com/dp/B0GF27J59X)

#### Hackathon Design
At at hackathon in early 2026, University of Washington's [CREATE](https://create.uw.edu/) and HuskyADAPT collaborated with the King County Library System to hold a mini-hackathon open to members of the community, which this project was a part of. There, participants developed a method for creating a bracelet extender of any length from three snap bracelets. Our goal at the GitHub OpenSource Assistive Technology hackathon was to improve documentation for this method. The bracelet_extender.md file is our documentation of this process.

Future work and todos include developing a single extender that can be adjusted as needed (for example, one prototype included a jewelry-chain style extender, with a movable clasp) instead of a custom single-length solution.

### Lanyard

#### Background
For educators and clinicians who work with individuals that use AAC symbols, having Q Charms on a wearable lanyard can be helpful to have them on-hand for use. A clinician requested a Q charms lanyard, so this project sought to help create one that would be sturdy and easy to use.

#### Hackathon Design
The baseplate_test.stl file is a ready-to-3D-print file developed during the GitHub hackathon event, for a wearable lanyard attachment that can have Q Charms slid onto it.

We were not able to print this out to test the fit of Q charms during the event, so the part may need slight modifications in terms of dimensions. Here is the [OnShape project](https://cad.onshape.com/documents/09468889b1cdf04ae84cd35e/w/854f4c16879e1fe97e757252/e/7917af0964783365681fc913?renderMode=0&uiState=6a10c590b4f90da79c44fc5f), which can be copied to make any necessary edits.

Future work and todos include making this double-sided to maximize the number of Q Charms able to be used with this item, and develop a way to address the plate potentially flipping around when worn on a lanyard with a single attachment point, as it is currently designed for.
