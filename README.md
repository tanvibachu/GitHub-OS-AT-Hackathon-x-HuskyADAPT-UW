# GitHub-OS-AT-Hackathon-x-HuskyADAPT-UW
Adaptable Fanny Pack Project

This project matters for individuals with disabilities as many people with developmental disabilities rely on small personal bags to carry essential items such as communication devices, credit/transit cards, money, keys, or medical information. Current fanny packs on the market have small zippers and poor internal organization, and relying on these market products might deal to struggle for disabled individuals to open them independently, locate items quickly, or return items using one hand. This can create frustration, increase dependence on caregivers, and make everyday tasks like paying for transportation, communicating, or making purchases more difficult. 


Background:
A student with developmental disabilities has a fanny pack without easy accessibility features to take out/put back items from the fanny pack. 

Problem:
The zippers are too small and can be hard to open, and since it’s a small bag, all the items inside the fanny pack get mixed around, and becomes hard to identify which item is which, and needs an organization system to be able to grab whatever the individual wants with ease.  

Aim:
Adapt a fanny pack to be able to easily access things when they are outside engaging in tasks such as communicating, purchasing, and when riding the bus. Make it accessible to grab objects with one hand.  ​​​

Design Criteria:
- Fanny pack's alterations must be lightweight and accessible to be able to open and close the fanny pack with one hand, while also being able to grab and place items with one hand. ​
- Set up a system that allows for easy organization when placing and removing items from fanny pack. ​

Suggestions:
- A tether cable attached to the fanny pack for easy card access when riding the bus. ​
- Color-coded lanyards for easy visual identification. ​
- Larger zipper loops.  ​
- Add a little pouch to store feminine products (don't attach inside fanny pack) ​
